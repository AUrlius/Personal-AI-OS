## 🚀 使用示例

### 基础使用
```python
from src.skills.core import SkillSystem, SkillBuilder

# 创建技能系统
skill_system = SkillSystem()

# 创建技能
skill = (SkillBuilder()
    .id("my-skill")
    .name("My Skill")
    .code("def execute(x): return {'result': x * 2}")
    .build())

# 注册并执行
await skill_system.register(skill)
result = await skill_system.execute("my-skill", {"x": 5})
```
